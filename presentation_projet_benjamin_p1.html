Présentation

Dans le cadre de la spécialité Informatique et Sciences du Numérique, j’ai, avec mes deux camarades, choisi comme projet la conception d’un jeu en python, ainsi que d’un site le présentant.

Nous souhaitions un site restant dans la thématique du jeu PACMAN in the MAZE (reprenant lui-même les graphismes du célèbre jeu PACMAN).
Dans un premier temps nous voulions un site compatible avec tous les écrans d’ordinateur (à partir de 1280px de largeur), puis, par la suite, nous avons voulu compléter cette démarche en l’adaptant à tout type de support, dont les mobiles et les tablettes.

Pour réaliser notre projet nous avons opté d’une part pour le langage PYTHON pour le jeu, et d’une autre pour les langages HTML5 et CSS3 pour le site.

Les programmes utilisés pour ce projet sont PYTHON pour le jeu et Sublime Text comme éditeur de texte.
