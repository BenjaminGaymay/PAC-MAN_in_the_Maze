JAVASCRIPT / JQUERY

Le lien de téléchargement du site et de son code est disponible → ici ←

Ce script JQuery n'est pas de moi. Mes camarades ont tenu à ce que le scroll entre les différents liens de la page d'accueil se fassent avec une transition. Je me suis donc penché du coté du javascript qui s'est avéré très compliqué à élaborer. Par la suite, par manque de temps j'ai choisis d'utiliser un script existant.

Afin de l'utiliser je dois importer la bibliothèque JQuery. Je place donc le code ci-dessus à la fin du BODY dans ma page HTML pour y avoir accès.
J'exécute ensuite le script permettant le "smooth scroll".

Le fonctionnement du script est le suivant :
- Quand on clique sur un lien dont le "href" se commence par "#" la variable "the_id" se créée et prend pour valeur l'attribut de "href"
- Puis on utilise l'animation "scrollTop" de JQuery pour aller lentement jusqu'à l'élément dont le nom est dans la variable "the_id", donc l'attribut de "href" (notre ancre)
