HTML

Du coté du site, j'ai entièrement codé le HTML ansi que le CSS.
Le lien de téléchargement du site et de son code est disponible → ici ←

Tout d'abord, on défini dans le "head" les caractéristiques de la page, mais elles ne sont pas visibles pour l’utilisateur.
"meta" permet d’annoncer au navigateur que les caractères sont en UTF-8.
Les deux "link" permettent de lier une page CSS nommée "style.css" et une icône (qui est affichée dans l’onglet du navigateur) à notre page html.
"title" est le nom de la page visible dans l’onglet du navigateur.

Le code HTML est très simple et se répète beaucoup, je vais donc uniquement commenter la partie traitant de la navigation ci-dessus, qui regroupe les éléments les plus intéressants.

Tout d'abord, "nav" est une balise intégrée en CCS3 désignant la navigation du site.

Afin de faciliter le positionnement des mes liens dans cette "nav" je les place sous forme de liste avec la balise "ul" qui indique ce qui fait parti de la liste, et des balises "li" qui désignent chacune des lignes de celle-ci.

Ensuite, la balise "a" me permet de créer un lien redirigeant vers le site, ou la page, de l'url indiqué par "href".

"img" est une image se trouvant à l'url défini par "src", et "h1" est texte de type grand titre.

"img" et "h1" se trouvant entre les balises "a" et "/a" sont tous deux un seul et même lien (celui défini précédemment par "href").

Vous pouvez également voir dans plusieurs de ces balises des "class" et des "id". Les "class" correspondent à des noms communs entre plusieurs éléments.

Par exemple, ici, toutes les images se nomment "p1_projet_img_nav" ce qui me permet par la suite, dans le CSS, de définir une seule fois leurs caractéristiques communes.

En revanche, la "nav" possède une "id" qui est un nom unique. Elle est la seule à se prénommer comme cela et je ne modifierai qu'elle dans le CSS.
