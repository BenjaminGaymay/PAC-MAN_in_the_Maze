Notre site web

Nous avons décidé de créer un site web référant notre jeu. Le site permet de donner un bref résumé de notre projet ainsi que les règles.
Le site de " PAC-MAN in the MAZE " est accessible sur internet, mais aussi à l'aide d'un lien disponible dans le launcher du jeu.
Ce site Internet offre la possibilité de télécharger la dernière version existante de notre projet.

Nous avons créé un site internet s'adaptant à tous types d'écran ainsi qu'à la majorité des smartphones.
Cette seconde caractéristique me tenait à cœur, parce qu'aujourd'hui, je trouve important le fait d'avoir la possibilité de consulter un site internet depuis son smartphone.
